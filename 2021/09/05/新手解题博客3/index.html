前置技能——HTTP协议

请求方式

题目：

解题：

猜想CTF**B请求方式为ctfhub

姿势1：burpsuite抓包

使用burpsuite进行抓包，发送到repeatr把get方式改为CTFHUB的请求方式，点击send，就会返回含有flag的数据包。如下图。（图来自https://blog.csdn.net/hgjiayou/article/details/106212933）

姿势2：curl命令

在cmd命令窗口中输入“curl -v -X CTFHUB http://challenge-61f16092c6f73cd5.sandbox.ctfhub.com:10800/index.php”即可得到flag信息

补充知识：

curl

cURL是一个利用URL语法在命令行下工作的文件传输工具。用来请求 Web 服务器。

学习链接：https://www.ruanyifeng.com/blog/2019/09/curl-reference.html
