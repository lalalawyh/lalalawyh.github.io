dirsearch安装与使用

简介

dirsearch是一个python开发的目录扫描工具，目的是扫描网站的敏感文件和目录从而找到突破口。

安装

进入官网下载https://github.com/maurosoria/dirsearch，点击代码code然后下载压缩包。

下载完成后解压。打开进入到dirsearch目录，cmd进入到dirsearch.py目录。可直接在这个目录下输入cmd进入，这样就不需要cd一级一级的进入了。

输入命令python dirsearch.py -u https://www.baidu.com -e *进行测试。

问题1：出现ModuleNotFoundError: No module named ‘chardet’

解决方法：手动进入网址https://github.com/chardet/chardet下载chardet模块。解压打开文件夹，cmd进入命令窗口输入命令python setup.py install进行安装。

问题2：出现ModuleNotFoundError: No module named ‘markupsafe’

解决方法：直接win+R召唤出命令窗口，输入命令pip install markupsafe进行安装。

没有问题之后再次进行测试就可以了。
