dvcs-ripper安装与使用

注：适用于Linux系统，windows用户可以安装在虚拟机上。

简介：

一款perl的版本控制软件信息泄露利用工具，支持SVN, GIT, Mercurial/hg, bzr…

安装：

在网址https://github.com/kost/dvcs-ripper上进行下载压缩包，解压后拖进虚拟机中。

进行简单配置相关要求的组件：sudo apt-get install perl libio-socket-ssl-perl libdbd-sqlite3-perl libclass-dbi-perl libio-all-lwp-perl

栗子

以ctfhub的svn泄露为栗子。

进入dvcs-ripper-master文件夹右键进入终端，输入命令perl rip-svn.pl -u http://challenge-277a34a7a99ae035.sandbox.ctfhub.com:10800/.svn/

再tree看看：根据提示安装：sudo apt install tree。然后输入命令：tree .svn进行查看

需要注意的是，.svn是隐藏文件，在linux下必须ls -al才能看到。
根据题目的提示，Flag 在服务端旧版本的源代码中。
注意tree中的pristine，里面一般存储的是代码的历史版本。直接进入目录，然后读取文件就可以得到flag了：

由于我在文件夹中没有看到那些文件夹，所以是一路cd进到be文件夹，再用cat命令打开文件查看flag。
