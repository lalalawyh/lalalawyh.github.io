VMware虚拟机网卡类型

  1. VMnet 0 （桥接模式）
       将主机网卡与虚拟机虚拟的网卡利用虚拟网桥进行通信。在桥接的作用下，类似于把物理主机虚拟为一个交换机，所有桥接设置的虛拟机连接到这个交换机的一个接口上，物理主机也同样插在这个交换机当中，所以所有桥接下的网卡与网卡都是交换模式的，相互可以访问而不干扰。
  2. VMnet 1 （仅主机模式）
    用于建立一个与世隔绝的网络环境所用到的，其中VMnet 1是一个虛拟的交换机，交换机的一个端口连接到你的主机上，另外一个端口连接到虚拟的DHCP服务器上（实际上是VMware的一个组件），另外剩下的端口就是连虚拟机了。
  3. VMnet 8 （NAT模式）
    最简单的组网方式，从主机的“VMware Virtual Ethernet Adapter for VMnet8”虚拟网卡出来，连接到VMnet8虛拟交换机，虚拟交换机的另外的口连接到虚拟的NAT服务器（这也是一个VMware组件），还有一个口连接到虚拟DHCP服务器，其他的口连虚拟机。

kali

kali的基本介绍

下载地址：https://www.kali.org/

Kali Linux是基于Debian的Linux发行版，设计用于数字取证操作系统。每一季度更新一次。由Offensive Security Ltd维护和资助。Kali Linux预装 了许多渗透测试软件，包括nmap、Wireshark 、John the Ripper，以及Aircrack -ng.用户可通过硬盘、live CD或live USB运行Kali Linux。Kali Linux既有32位和64位的镜像。

基本守则

Linux系统大小写敏感。
文件名前面带”.”是隐藏文件。
Linux 系统下的“/”相当于Windows 下的“\”。
配置文件中，#开头代表是注释行，程序不执行该行。

基本命令

pwd //显示当前所在的目录位置
Is【参数】【对象】 //显示当前目录或指定目录中的文件
●-a //显示所有文件(包含隐藏文件)
●-| //显示文件与目录的详细信息
cd
●cd dir //进入一个目录
●cd.. //进入当前目录的上一-级目录.
●cd- //返回进入此目录之前所在目录
●cd //返回桌面

touch file/vi file //创建文件
●Touch是立即生成，vi必须要保存才能生成相应的文件
mkdir目录 //创建目录
rmdir目录 //删除目录，但只能删除空目录
rm [参数] //删除文件
●-i //确认信息，删除文件或者目录前会进行一-次确认
●-r //递归，删除目录时必须使用
●-f //强制删除，不提示和确认

cp [参数] sou dst //复制文件（默认）或目录（需要加参数）
●-p //保留原来的时间戳、拥有者信息等
●-r //复制目录
●-a //等同于同时使用-rp
mv [参数] file dst移动、重命名文件或目录
●dst仅是文件名，重命名
●dst是目录和文件名，但目录不是当前目录，移动并重命名
●dst仅是目录，此时是移动文件，此时可以移动多个文件
find [path] [expr]在路径path中查找符合表达式expr的文件。
●-name goktech //文件名符合goktech的文件（加上-i忽略大小写，如find/-i name goktech）

数据导出
●> 导出（覆盖）echo ‘1’ > goktech.txt
●>> 导出（追加） echo ‘2’ >> goktech.txt
cat显示文件的所有内容
●-A显示换行字符；
●-η显示行号；
●-b：和-n相似，但对空白行不编号；

文本编辑vi/vim

三种模式
●命令模式：可移动光标位置，可修改、删除、复制、粘贴、搜索关键字。
●插入模式：输入新的字符，点esc键回到命令模式。
●扩展模式：命令模式下输入“:”进入扩展模式，保存、退出、搜索等。
命令模式进入插入模式
●a光标会跳到当前后一格位置
●i 光标位置不变
●A光标会停在当前行的最后位置
●1光标会停在当前行的最前面的位置
●o会在光标所在行的下面新增一行空白
●0会在光标所在行的上面新增一 行空白
●r替换光标所在位置的字符，但不进入插入模式
●R从光标所在位置开始替换，能替换多个字符

扩展模式的命令
●:W 保存
●:w! 强制保存
●:q 退出
●:q! 强制离开
●:wq 保存及退出
●:wq! 强制保存离开
