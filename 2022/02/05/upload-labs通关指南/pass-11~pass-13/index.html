upload-labs靶场

pass-11

解题：（双写绕过）

        $deny_ext = array("php","php5","php4","php3","php2","html","htm","phtml","pht","jsp","jspa","jspx","jsw","jsv","jspf","jtml","asp","aspx","asa","asax","ascx","ashx","asmx","cer","swf","htaccess","ini");

        $file_name = trim($_FILES['upload_file']['name']);//$_FILES['upload_file']['name']获取上传文件的名称，trim($_FILES['upload_file']['name'])删除文件名称首尾的空白符
        $file_name = str_ireplace($deny_ext,"", $file_name);//删除变量$file_name中含有数组$deny_ext的字符串
        $temp_file = $_FILES['upload_file']['tmp_name'];//在上传文件的时候，文件都会被存储在一个临时的文件夹下，我们不需要知道具体路径，只需要通过tmp_name参数获取路径即可
        $img_path = UPLOAD_PATH.'/'.$file_name;//UPLOAD_PATH的值在config.php文件中定义

可以构造一个后缀为.pphphp的一句话木马文件，因为置空的操作只执行一次，所以删除了一次中间的php字符串后，p与hp仍然组成了.php后缀，不可以改为.phpphp会被全删掉

pass-12

解题：（白名单GET型00截断绕过=文件路径%00截断）

    $ext_arr = array('jpg','png','gif');
    $file_ext = substr($_FILES['upload_file']['name'],strrpos($_FILES['upload_file']['name'],".")+1);//获取文件的后缀名，strrpos($_FILES['upload_file']['name'],"."获取小数点右移一位的字符下标
    if(in_array($file_ext,$ext_arr)){
        $temp_file = $_FILES['upload_file']['tmp_name'];
        $img_path = $_GET['save_path']."/".rand(10, 99).date("YmdHis").".".$file_ext;
        //save_path的值是通过GET的方式传入；$file_ext是文件的后缀，不含小数点；rand(10,99)则是从10到99数字中取随机数，范围左闭右闭；date("YmdHis")用于获取当前的时间，以xxxx年xx月xx日xx时xx分xx秒为格式，两个字符串变量之间的连接用小数点
        if(move_uploaded_file($temp_file,$img_path)){
            $is_upload = true;
        }

上传白名单后缀.jpg，例如文件pass12.jpg，抓包修改Raw中save_path的值为../upload/pass12.php%00，复制图片地址后，发现后面有些没用的字符信息，直接删除.php之后的内容进行访问，访问成功表示文件被解析了

小知识：

00绕过需要有两个条件：（1）php.ini文件中的magic_quotes_gpc状态为off，（2）php的版本要小于5.3.4

pass-13

解题：（文件路径二进制00截断）

$img_path = $_POST['save_path']."/".rand(10, 99).date("YmdHis").".".$file_ext;

与pass12相比，save_path的值是通过POST方式传入的。POST不会像GET对%00进行自动解码，所以需要在二进制中进行修改。即利用Burpsuite的Hex功能将save_path改成…/upload/12.php【二进制00】形式
